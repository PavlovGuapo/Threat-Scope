Dashboard de inteligencia de vulnerabilidades. Permite procesar archivos XML generados por herramientas de seguridad (Nmap, OWASP ZAP, BurpSuite, OpenVAS, Nessus) para centralizar la gestión de hallazgos, visualizar métricas y generar reportes automatizados en formato Word (.docx).

## Instalación y Despliegue

El proyecto está dockerizado para facilitar su ejecución. Asegúrate de tener Docker y Docker Compose instalados.

1. Clona este repositorio:
   git clone https://github.com/tu-usuario/threat-scope.git

2. Ingresa al directorio:
   cd threat-scope

3. Construye y levanta el contenedor en segundo plano:
   docker compose up -d 

La aplicación estará disponible en tu navegador apuntando a `http://localhost:3000`.

## Persistencia de Datos

Toda la información almacenada, incluyendo el estado de las vulnerabilidades y las imágenes de evidencia fotográfica, se mantiene segura en un volumen Docker persistente. Si necesitas borrar la base de datos y reiniciar la herramienta desde cero, ejecuta `docker compose down -v`.
